IoT and Electronics Instrumentation

  1. Microcontroller-Based Automatic Security System Using Pin(trought mobile app)and Fingerprints

    2. Concept and How the Tool Works: Controlled the turnstile lock using the enrolled fingerprint sensor. The fingerprint sensor would only change the state of the fence lock if the confidence level of the detected fingerprint was more than 5. The lock could also be controlled through an application made with Kodular with a real time database using Firebase. To control with the application, the user must enter a pin password. The state of the gate was open or closed can also be notified to the application with a push button as a detector. Battery management system was also used to prevent the lock cannot be controlled when there was a power outage.

  2. SMART ROOM

    3. Concept and How the Tool Works: Turning the lights on and off automatically. If someone entered the room, the infrared sensor would detect the movement and sent the data to the relay which became a counter and counted 1, after which the lights turned on. If the person left and there was no body in the room, the infrared sensor would detect and the counter would count 0, after which the lights would turn off.

  3. Reverse Vending Machine Plastic Bottle Waste

    4. Concept and How the Tool Works: Could issue a coupon printed using a thermal printer when a plastic bottle was inserted into this tool, the number of vouchers would also be displayed on the LCDI2C screen. The amount of vouchers obtained depends on the colour of the bottle which could be detected by the TCS3200 sensor. If the bottle inserted contained water then the bottle would be rejected, to detect it, a loadcell sensor was used. Rejected bottles would be removed again using a servo motor and those that pass would be put into the box.

  4. Water Level Control System for Hydroponic Farming Using PID Method

    5. Concept and How the Tool Works: controlled the water level in hydroponic pots so that the water level was always in accordance with what had been determined. the water level would be measured using an ultrasonic sensor and then the measurement results would be processed with an arduino microcontroller that had been programmed using the PID method. If the water level was less than the specified then the pump would add water and if the excess valve at the end of the pot would open. The water level could be changed using a potentiometer and the water level would be displayed on LCDI2C in real-time.

  5. Blynk's IoT-Based Automatic Light

    6. Concept and How the Tool Works: Turned lights on and off remotely using the Blynk app. In the Blynk app people could just press the On and Off button from any distance, as long as the device was connected to the internet.

  6. IoT-Based Automatic Fish Feeder

    7. Fed the fish every 6 hours by opening the valve stored at the bottom of the device. This valve was controlled using a servo motor and would open for 10 seconds until it closed again
